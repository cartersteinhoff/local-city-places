# Merchant Dashboard Implementation Plan

## Overview

Complete the merchant dashboard with 6 main sections:
1. **Dashboard** - Real stats, activity feed, inventory overview
2. **Issue GRC** - Individual + bulk GRC issuance
3. **My GRCs** - List all issued GRCs with status tracking
4. **Surveys** - Create and manage customer surveys
5. **Reviews** - View member reviews
6. **Profile** - Already complete

---

## Current State

**Existing:**
- `/merchant` - Basic dashboard (hardcoded stats)
- `/merchant/profile` - Fully functional
- Navigation defined in `nav.ts`
- Database schema ready (grcs, grc_purchases, surveys, reviews tables)

**Missing:**
- `/merchant/issue` - Issue GRC page
- `/merchant/issue/bulk` - Bulk upload page
- `/merchant/grcs` - GRC list page
- `/merchant/surveys` - Survey management
- `/merchant/surveys/new` - Create survey
- `/merchant/surveys/[id]` - Edit survey
- `/merchant/reviews` - Reviews list
- All merchant API routes

## API Routes

### Dashboard Stats
```
GET /api/merchant/dashboard
Response:
{
  stats: {
    activeGrcs: number,
    redeemedGrcs: number,
    activeMembers: number,
    avgRating: number | null,
    totalReviews: number
  },
  inventory: [
    { denomination: 50, available: 23, total: 50 },
    { denomination: 100, available: 12, total: 50 }
  ],
  recentActivity: [
    { type: 'registration', memberName: 'John D.', grcValue: 100, date: '...' },
    { type: 'completion', memberName: 'Jane S.', month: 2, totalMonths: 4, date: '...' },
    { type: 'review', memberName: 'Mike R.', rating: 5, date: '...' }
  ]
}
```

### GRC Management
```
GET /api/merchant/grcs?status=all&search=&page=1&limit=10
Response:
{
  grcs: [
    { id, email, recipientName, denomination, status, monthsCompleted, totalMonths, issuedAt }
  ],
  pagination: { total, page, limit, totalPages }
}

POST /api/merchant/grcs/issue
Body: { email, recipientName?, denomination }
Response: { success, grc: { id, claimUrl } }

POST /api/merchant/grcs/bulk
Body: { recipients: [{ email, name?, denomination }] }
Response: { success, issued: number, errors: [{ row, message }] }
```

### Surveys
```
GET /api/merchant/surveys
Response: {
  surveys: [
    { id, title, questionCount, responseCount, isActive, createdAt, lastResponseAt }
  ]
}

POST /api/merchant/surveys
Body: {
  title: string,
  questions: [
    { text: string, type: 'text' | 'multiple_choice', options?: string[], required: boolean }
  ]
}

GET /api/merchant/surveys/[id]
Response: { survey: { id, title, questions, isActive, createdAt } }

PUT /api/merchant/surveys/[id]
Body: { title?, questions?, isActive? }

DELETE /api/merchant/surveys/[id]

GET /api/merchant/surveys/[id]/responses
Response: {
  survey: { id, title, questions },
  responses: [
    { id, memberId, memberName, answers: { questionId: answer }, createdAt }
  ],
  summary: {
    questionId: {
      type: 'multiple_choice',
      options: { optionText: count }
    } | {
      type: 'text',
      answers: string[]
    }
  }
}
```

### Reviews
```
GET /api/merchant/reviews?page=1&limit=10
Response: {
  stats: { avgRating, totalReviews, positivePercent },
  reviews: [
    { id, memberName, content, wordCount, rating?, createdAt }
  ],
  pagination: { total, page, limit, totalPages }
}
```

## Database Queries

### Dashboard Stats
```sql
-- Active GRCs (status = 'active')
SELECT COUNT(*) FROM grcs WHERE merchant_id = ? AND status = 'active';

-- Redeemed GRCs (status = 'completed')
SELECT COUNT(*) FROM grcs WHERE merchant_id = ? AND status = 'completed';

-- Active Members (distinct members with active GRCs)
SELECT COUNT(DISTINCT member_id) FROM grcs WHERE merchant_id = ? AND status = 'active';

-- Average Review Score
SELECT AVG(rating) FROM reviews WHERE merchant_id = ?;

-- Inventory by denomination
SELECT denomination,
       SUM(quantity) as purchased,
       (SELECT COUNT(*) FROM grcs WHERE merchant_id = gp.merchant_id AND denomination = gp.denomination) as issued
FROM grc_purchases gp
WHERE merchant_id = ? AND payment_status = 'confirmed'
GROUP BY denomination;
```

### Recent Activity
```sql
-- Recent registrations
SELECT m.first_name, m.last_name, g.denomination, g.created_at
FROM grcs g
JOIN members m ON g.member_id = m.id
WHERE g.merchant_id = ? AND g.member_id IS NOT NULL
ORDER BY g.created_at DESC
LIMIT 5;

-- Recent reviews
SELECT m.first_name, m.last_name, r.rating, r.created_at
FROM reviews r
JOIN members m ON r.member_id = m.id
WHERE r.merchant_id = ?
ORDER BY r.created_at DESC
LIMIT 5;
```
